Crypto Price Prediction 

A Flask-based web application that predicts cryptocurrency prices using a trained deep learning (LSTM) model.

Features
 * Predicts future crypto prices
 * Simple and interactive web interface
 * Uses real-time data from yfinance
   
Tech Stack
 * Python
 * Flask
 * TensorFlow / Keras
 * Pandas, NumPy, Scikit-learn

Project Structure
 * app.py → Main Flask application
 * model.keras → Trained ML model
 * templates/ → HTML files
 * static/ → CSS/JS files
